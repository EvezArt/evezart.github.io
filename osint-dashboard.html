About EVEZ OSINT

EVEZ OSINT v1.1.0 — Suspect Inference Matrix Engine

Isolates and identifies suspect inference networking matrices. Computes statistical probabilities of likely crimes using eigenforensic spectral analysis.

AEMDAS Pipeline (6 stages = 6 cube faces):
1. ASSERT — Register suspects and events
2. EXTRACT — Build adjacency matrix
3. MEASURE — Compute eigenvalues and spectral gaps
4. DEDUCE — Infer crime hypotheses
5. ASSESS — Calculate probabilities and confidence
6. SPEEDRUN — Export reports

Features:
- 5 crime types: excessive force, civil rights, cover up, evidence tampering, false arrest
- Pure Python (no numpy required) — eigenvalue computation via power iteration
- SQLite database collects anonymous usage statistics
- Interactive CLI with colored output and progress bars
- JSON + Markdown export
- All probabilities falsifiable and bounded: η* ≤ P ≤ 0.95
